# Enterprise Performance Analytics Dashboard -- Cross-functional-Interactive-dashboard
Developed an interactive, cross-functional Power BI dashboard powered by SQL to provide unified insights across Finance, Sales, Marketing, and Supply Chain departments. The project enables data-driven decision-making by integrating and visualizing complex datasets for comprehensive business analysis.
